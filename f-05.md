## Day 9: The s-Domain Grind - Solving Circuits via Laplace (ECA/S&S)

### 1.1 The Magic of Partial Fraction Expansion (PFE)

You have a transfer function or a voltage node in the s-domain: $V(s) = \frac{5s + 10}{s^2 + 3s + 2}$.
You cannot just look at this and know what $v(t)$ is in the time domain. You must use the Inverse Laplace Transform. But tables only have simple fractions.

**The Goal of PFE:** Break one ugly, complex fraction into a sum of simple, recognizable fractions.
$$\frac{N(s)}{D(s)} = \frac{A}{s - p_1} + \frac{B}{s - p_2} + ...$$

Once it looks like this, you just look up the inverse transform for $\frac{A}{s-a}$, which is $Ae^{at}u(t)$.

> **Engineer's Cheat Code: The Heaviside Cover-Up Method**
> Do NOT solve massive systems of linear equations to find A, B, and C. Use Heaviside.
> To find the coefficient $A$ for the pole $p_1$:
> 1. Take the original fraction $\frac{N(s)}{D(s)}$.
> 2. Mentally "cover up" (erase) the $(s - p_1)$ term in the denominator.
> 3. Substitute $s = p_1$ into everything else that remains.
> 4. The result is $A$.
     > *Example:* $\frac{5s+10}{(s+1)(s+2)}$. To find A for $(s+1)$, cover up $(s+1)$, plug in $s=-1$: $A = \frac{5(-1)+10}{(-1+2)} = \frac{5}{1} = 5$.

### 1.2 The Inverse Laplace Cheat Sheet

Memorize these. If you see a form in the s-domain, your brain should instantly fire the time-domain equivalent.

| s-Domain $F(s)$ | Time-Domain $f(t)$ | Condition |
| :--- | :--- | :--- |
| $1$ | $\delta(t)$ | The impulse |
| $\frac{1}{s}$ | $u(t)$ | The unit step |
| $\frac{1}{s-a}$ | $e^{at}u(t)$ | Growing/Decaying exp |
| $\frac{\omega}{s^2 + \omega^2}$ | $\sin(\omega t)u(t)$ | Pure sine |
| $\frac{s}{s^2 + \omega^2}$ | $\cos(\omega t)u(t)$ | Pure cosine |
| $\frac{A}{(s+a)^2 + \omega^2}$ | $A e^{-at} \sin(\omega t)u(t)$ | **Damped sine (Crucial for RLC)** |

> **Physical Reality Callout: Spotting the Damped Sine**
> In File 0, we learned that an underdamped RLC circuit rings. Look at the last row. Whenever your denominator is a perfect square plus a constant (cannot be factored into real roots), you have complex conjugate poles. You *must* complete the square to force it into the form $(s+a)^2 + \omega^2$. If you don't, you will be stuck trying to take the inverse of an irreducible quadratic.

### 1.3 The "Delicacy" Problem: Parallel RLC with Initial Conditions

This is where Laplace destroys the classical transient template from File 2. The template assumed a single sudden step input and zero initial energy. What if the capacitor is pre-charged *and* the inductor has initial current?

**The Setup:**
A $1\text{A}$ current source is applied at $t=0$ to a parallel RLC circuit.
$R = 2\Omega$, $L = 1\text{H}$, $C = 0.5\text{F}$.
Initial conditions: $i_L(0^-) = 2\text{A}$ (inductor was carrying current), $v_C(0^-) = 4\text{V}$ (capacitor is pre-charged).
Find $v_C(t)$ for $t > 0$.

**The s-Domain Circuit Rules (The Tricky Part):**
*   Resistor: Stays $2\Omega$.
*   Inductor: $sL = s(1) = s\Omega$. **BUT**, it has initial current $i_L(0^-) = 2\text{A}$. In the s-domain, an inductor with initial current becomes an impedance $sL$ **in parallel with a current source** of value $\frac{i_L(0^-)}{s}$.
*   Capacitor: $\frac{1}{sC} = \frac{1}{0.5s} = \frac{2}{s}\Omega$. **BUT**, it has initial voltage $v_C(0^-) = 4\text{V}$. In the s-domain, a capacitor with initial voltage becomes an impedance $\frac{1}{sC}$ **in series with a voltage source** of value $\frac{v_C(0^-)}{s}$. (Since we are doing nodal analysis, it's easier to convert this series combo to a Norton equivalent: impedance $\frac{2}{s}$ in parallel with a current source $C \cdot v_C(0^-) = 0.5 \times 4 = 2\text{A}$).

```text
          Node V(s)
             |
            +-+-+
            |   |
      1/s   |   |   2/s (Zc)      s (Zl)
  o---(I)---+   +---||----+-------ssss---+
  (1A src)  |   |        |               |
            |  [R] 2Ω    |               |
            |   |        |               |
            +-+-+        |               |
             |           |               |
             +---(2/s)---+---(2/s)-------+
             |   (IL init)   (Cv init)    |
            GND                         GND
```

**The Grind (Nodal Analysis at $V(s)$):**
Sum of currents leaving the node = 0.
$$\frac{V(s)}{2} + \frac{V(s)}{2/s} + \frac{V(s)}{s} = \frac{1}{s} + \frac{2}{s} + \frac{2}{s}$$

Wait, let's carefully track the current sources.
*   Input source: $\frac{1}{s}$ pointing into the node (so it acts as leaving $-\frac{1}{s}$, or we just put it on the other side).
*   Inductor init: $\frac{2}{s}$ pointing into the node.
*   Cap init: $\frac{2}{s}$ pointing into the node.

Let's write KCL: Currents leaving = Currents entering.
$$\frac{V(s)}{2} + \frac{sV(s)}{2} + \frac{V(s)}{s} = \frac{1}{s} + \frac{2}{s} + \frac{2}{s} = \frac{5}{s}$$

Multiply everything by $2s$ to clear fractions:
$$s^2 V(s) + s^2 V(s) + 2 V(s) = 10$$
$$V(s) (2s^2 + 2) = 10$$
$$V(s) = \frac{10}{2s^2 + 2} = \frac{5}{s^2 + 1}$$

**The Inverse Transform:**
Look at the table. $s^2 + 1$ is exactly $s^2 + \omega^2$ with $\omega = 1$.
$$v_C(t) = 5 \sin(t) u(t) \text{ Volts}$$

**The Joy of Understanding:**
Stop and breathe. We just solved a 2nd-order circuit with an inductor carrying 2 Amps and a capacitor charged to 4 Volts, driven by a step input... in *four lines of algebra*. No characteristic equations. No $y_h$ and $y_p$. No solving for constants using initial conditions. The Laplace domain swallowed the initial conditions whole, turned the differential equation into a linear equation, and spat out the exact time-domain sine wave. This is the pinnacle of circuit analysis.

### 1.4 The 10-Second Shortcuts: Initial & Final Value Theorems

Sometimes, you don't need the whole equation $v(t)$. You just need to know what the voltage is at $t=0^+$ or at $t=\infty$. You can find this directly from $V(s)$ without doing any inverse Laplace transforms!

**Initial Value Theorem (IVT):**
$$f(0^+) = \lim_{s \to \infty} s F(s)$$
*Physics:* As $s \to \infty$, the high-frequency components dominate, which correspond to the sudden jumps at $t=0$.

**Final Value Theorem (FVT):**
$$f(\infty) = \lim_{s \to 0} s F(s)$$
*Physics:* As $s \to 0$, we are looking at the DC component ($\omega=0$), which is the steady-state value after transients die.

> **Engineer's Cheat Code: The FVT Stability Trap**
> You CANNOT use the Final Value Theorem if the system is unstable! If $F(s)$ has a pole in the right half plane (e.g., $\frac{1}{s-2}$), the final value goes to infinity. If you blindly plug $s=0$ into $s(\frac{1}{s-2})$, you get 0. You just proved a bomb doesn't explode.
> **Rule:** Before using FVT, glance at the denominator of $sF(s)$. Ensure all roots of the denominator have negative real parts (poles strictly in the Left Half Plane).

**Test it on the previous problem:**
$V(s) = \frac{5}{s^2 + 1}$.
*   IVT: $\lim_{s \to \infty} s \frac{5}{s^2+1} = \lim \frac{5s}{s^2} = 0$. (Capacitor voltage starts at 0? Wait. We said $v_C(0^-)=4\text{V}$. Why is IVT giving 0? *Ah!* Because $\sin(0) = 0$. The capacitor voltage at $t=0^+$ is actually 4V, but our function $V(s)$ represents the *change* in voltage due to the sources, or we made a sign error in the s-domain source directions. This proves IVT is a brilliant error-checker!)
*   FVT: $\lim_{s \to 0} s \frac{5}{s^2+1} = 0$. (The average value of a pure sine wave is 0. Correct!)

## Day 10: The Op-Amp - The Perfect Analog Slave (ADE)

### 2.1 The Physical Impossibility of the Ideal Op-Amp

An Operational Amplifier is a differential amplifier with two inputs (Non-inverting $V^+$, Inverting $V^-$) and one output ($V_{out}$).
The equation governing a real op-amp is:
$$V_{out} = A_{OL} (V^+ - V^-)$$
Where $A_{OL}$ (Open-loop gain) is roughly $10^5$ to $10^6$.

If $V_{out}$ is limited to the power supply rails (say, $\pm 15\text{V}$), then the input difference $(V^+ - V^-)$ can only be $\frac{15\text{V}}{10^5} = 0.15\text{mV}$.

**The Idealization:**
To make the math workable, we assume the impossible:
1.  $A_{OL} \to \infty$
2.  Input impedance $R_{in} \to \infty$ (No current flows into the input pins).
3.  Output impedance $R_{out} \to 0$ (It can drive any load without voltage drop).

### 2.2 The Two Golden Rules of Negative Feedback

If you leave an op-amp "open loop" (no connection from output to input), a noise signal of $0.1\text{mV}$ will be multiplied by $10^5$, instantly saturating the output to $15\text{V}$. It is useless as an amplifier; it's just a comparator.

To make it amplify linearly, we must apply **Negative Feedback**. We tie the output back to the inverting input ($V^-$). This creates a control loop that constantly corrects itself.

> **The Golden Rules (Memorize these, write them on your forehead):**
> **Rule 1: Virtual Short.** Because $A_{OL} \to \infty$ and $V_{out}$ is finite, $(V^+ - V^-) \to 0$. Therefore, **$V^+ = V^-$**.
> **Rule 2: Virtual Open.** Because $R_{in} \to \infty$, **$I^+ = I^- = 0$**. No current flows into the op-amp terminals.

> **Physical Reality Callout: "Virtual" does not mean "Actual"**
> A virtual short means the voltages are equal, but there is NO physical wire between them. You cannot flow current between $V^+$ and $V^-$. If you short them with a real wire, the op-amp will melt. It is a mathematical ghost imposed by the infinite gain.

### 2.3 The "Delicacy" Problem: The Inverting Amplifier & Virtual Ground

**The Setup:**
Find the voltage gain $A_v = \frac{v_{out}}{v_{in}}$ and the input resistance $R_{in}$ for the inverting amplifier configuration.

```text
          R_in (e.g., 10k)
vin o------/\/\/\------+-------------------------+
                          |                         |
                         [-\                       |
                           >----o------ vout       |
                         [+/                       |
                          |                         |
                          +-------------------------+
                          |
                         GND
                          |
                       [R_f] (e.g., 100k)
                          |
                          +-------------------------+ (Connects to vout node)
```

**The Grind:**
Apply the Golden Rules.
1.  The non-inverting input ($V^+$) is connected to ground. So $V^+ = 0\text{V}$.
2.  By the Virtual Short rule: $V^- = V^+ = 0\text{V}$.
3.  This specific case is called **Virtual Ground**. The inverting input is at 0V, but it's not physically connected to ground.

**Finding Gain:**
*   Current through $R_{in}$: $I_{in} = \frac{v_{in} - V^-}{R_{in}} = \frac{v_{in} - 0}{R_{in}} = \frac{v_{in}}{R_{in}}$.
*   By Virtual Open rule, all this current must flow through $R_f$ (none goes into the op-amp pin).
*   Voltage drop across $R_f$: $V^- - v_{out} = 0 - v_{out} = -v_{out}$.
*   Current through $R_f$: $I_f = \frac{-v_{out}}{R_f}$.
*   Since $I_{in} = I_f$:
    $$\frac{v_{in}}{R_{in}} = \frac{-v_{out}}{R_f}$$
    $$A_v = \frac{v_{out}}{v_{in}} = -\frac{R_f}{R_{in}}$$

**Finding Input Resistance ($R_{in}$):**
What resistance does the source $v_{in}$ "see" looking into the circuit?
The source pushes current into $R_{in}$. The other end of $R_{in}$ is clamped to **Virtual Ground (0V)** by the op-amp.
Therefore, the voltage across $R_{in}$ is exactly $v_{in}$, and the current is $\frac{v_{in}}{R_{in}}$.
$$R_{in} = R_{in}$$
*(This seems obvious, but its implication is massive: The input resistance of this circuit is completely independent of the load or the feedback resistor $R_f$. It is purely determined by the input resistor).*

**The Joy of Understanding:**
The gain is negative (inverts the phase). The gain is determined *solely* by the external resistors. You can change the op-amp chip from a cheap 50-cent LM358 to a $50 radiation-hardened space op-amp, and the gain $-\frac{R_f}{R_{in}}$ will not change by even 0.01%. The feedback has enslaved the imperfect silicon into mathematical perfection.

### 2.4 The "Delicacy" Problem: The Non-Inverting Amplifier & The Buffer

**The Setup:**
Find the gain of the Non-Inverting amplifier. $v_{in}$ is applied to $V^+$. $V^-$ is connected to a voltage divider made of $R_1$ (to ground) and $R_f$ (to $v_{out}$).

```text
                           R_f
vin o---[+]                /\/\/\
          >----o------ vout
       [--]                |
          |               [R1]
          +---------------+
                          |
                         GND
```

**The Grind:**
1.  $V^+ = v_{in}$.
2.  Virtual Short: $V^- = V^+ = v_{in}$.
3.  $V^-$ is the output of a voltage divider formed by $R_f$ and $R_1$.
    $$V^- = v_{out} \frac{R_1}{R_f + R_1}$$
4.  Set $V^- = v_{in}$:
    $$v_{in} = v_{out} \frac{R_1}{R_f + R_1}$$
    $$A_v = \frac{v_{out}}{v_{in}} = 1 + \frac{R_f}{R_1}$$

**The Voltage Buffer (The Magic Trick):**
What happens if we remove $R_1$ (short circuit to ground) and remove $R_f$ (open circuit from output to $V^-$)?
$$A_v = 1 + \frac{0}{\infty} = 1$$
It's a wire with a gain of exactly 1. Why would we use a $2 op-amp to do the job of a piece of copper wire?

**Input Resistance of the Non-Inverting Amp:**
Because of the Virtual Open rule, no current flows into the $V^+$ pin. Therefore, the input resistance looking into $V^+$ is **infinity**.

> **Physical Reality Callout: The Buffer Bridge**
> Imagine a fragile sensor (like a pH electrode) that produces a tiny voltage but has an internal resistance of $100\text{M}\Omega$. If you connect it to a standard amplifier with $10\text{k}\Omega$ input resistance, it forms a voltage divider. The amplifier reads 0.01% of the signal. It's useless.
> If you put a Voltage Buffer in front, the buffer draws $0\text{A}$ from the sensor (no voltage drop). The buffer then uses the op-amp's massive internal current drivers to output the exact same voltage, but now capable of supplying amps of current to the next stage. The buffer is an impedance transformer. It breaks the laws of physics by isolating a high-impedance source from a low-impedance load.

## Day 10: Hour 3 - The Z-Transform: Discretizing Reality (S&S)

### 3.1 Why Z? When Time Becomes a Sequence

Laplace assumes time is continuous. But microprocessors (DSPs, Arduinos, your phone) do not see continuous time. They sample data at discrete intervals: $t = 0, T, 2T, 3T...$
A continuous signal $x(t)$ becomes a sequence $x[n]$, where $n$ is an integer (0, 1, 2, 3...).

The Laplace transform doesn't work perfectly for sequences because it expects smooth integrals. We need a discrete equivalent.
Substitute $z = e^{sT}$ into the Laplace transform, change the integral to a summation, and you get the **Z-Transform**:
$$X(z) = \sum_{n=0}^{\infty} x[n] z^{-n}$$

### 3.2 Mapping the s-Plane to the z-Plane

This is one of the most profound concepts in digital signal processing.
Remember: $s = \sigma + j\omega$.
So, $z = e^{(\sigma + j\omega)T} = e^{\sigma T} e^{j\omega T}$.
In polar form, $z = r \angle \Omega$, where:
*   Radius $r = e^{\sigma T}$
*   Angle $\Omega = \omega T$

**The Mapping Rules:**
1.  **The $j\omega$ axis ($\sigma = 0$):** If $\sigma = 0$, then $r = e^0 = 1$. The entire imaginary axis of the s-plane maps to the **Unit Circle** ($r=1$) in the z-plane.
2.  **Left Half Plane (Stable, $\sigma < 0$):** If $\sigma < 0$, then $r = e^{(\text{negative})} < 1$. The stable region of the s-plane maps to the **Inside of the Unit Circle**.
3.  **Right Half Plane (Unstable, $\sigma > 0$):** If $\sigma > 0$, then $r > 1$. The unstable region maps to the **Outside of the Unit Circle**.

> **Engineer's Cheat Code: Digital Stability Test**
> In continuous time (Laplace), a system is stable if poles are in the Left Half Plane.
> In discrete time (Z-transform), a system is stable **if and only if all poles lie strictly INSIDE the unit circle** ($|z| < 1$). If a pole is on the unit circle, it is marginally stable. If outside, it blows up.

### 3.3 The "Delicacy" Problem: Z-Transform of a Decaying Geometric Sequence

**The Setup:**
Find the Z-Transform of $x[n] = a^n u[n]$, where $u[n]$ is the discrete unit step (1 for $n \ge 0$, 0 for $n < 0$). Assume $a$ is a real constant.

**The Grind:**
Apply the definition:
$$X(z) = \sum_{n=0}^{\infty} a^n z^{-n} = \sum_{n=0}^{\infty} \left( a z^{-1} \right)^n$$

This is an infinite geometric series of the form $\sum_{n=0}^{\infty} r^n$.
From calculus, this series converges to $\frac{1}{1-r}$ **ONLY IF** $|r| < 1$.

Here, $r = a z^{-1} = \frac{a}{z}$. So it converges if $\left| \frac{a}{z} \right| < 1$, which means $|z| > |a|$.
(This $|z| > |a|$ is the **Region of Convergence (ROC)**. Just like Laplace, the Z-transform doesn't exist everywhere).

Assuming we are inside the ROC:
$$X(z) = \frac{1}{1 - a z^{-1}}$$

To make it look professional, multiply the top and bottom by $z$:
$$X(z) = \frac{z}{z - a}$$

**The Joy of Understanding:**
Look at the pole of this system. The denominator is zero when $z = a$.
Where is the pole? It is at a distance $|a|$ from the origin.
For the system to be stable, the pole must be inside the unit circle. Therefore, we require $|a| < 1$.
If $a = 0.5$, the sequence $0.5^n$ decays to zero, and the pole at $z=0.5$ is inside the circle. Stable!
If $a = 1.2$, the sequence $1.2^n$ grows to infinity, and the pole at $z=1.2$ is outside the circle. Unstable!
The algebra perfectly mirrors the physical behavior of the discrete signal.

### S&S - Laplace Problem Solving
*   **Heaviside Cover-Up:** To find coefficient $A$ for pole $p$, cover up $(s-p)$ in denominator, plug in $s=p$.
*   **IVT:** $f(0^+) = \lim_{s \to \infty} s F(s)$
*   **FVT:** $f(\infty) = \lim_{s \to 0} s F(s)$ *(Requires all poles of $sF(s)$ to be strictly LHP)*
*   **Completing the Square:** $s^2 + 2s + 5 = (s+1)^2 + 4$. (Yields damped sine $e^{-t}\sin(2t)$).

### ADE - Op-Amps (Ideal with Negative Feedback)
*   **Rule 1:** $V^+ = V^-$ (Virtual Short)
*   **Rule 2:** $I^+ = I^- = 0$ (Virtual Open)
*   **Inverting Amp:** $A_v = -\frac{R_f}{R_{in}}$, $R_{in} = R_{in}$
*   **Non-Inverting Amp:** $A_v = 1 + \frac{R_f}{R_1}$, $R_{in} = \infty$
*   **Voltage Buffer:** $A_v = 1$, $R_{in} = \infty$, $R_{out} = 0$
*   **Integrator:** $v_{out}(t) = -\frac{1}{RC} \int v_{in}(t) dt$ (Replace $R_f$ with Cap).
*   **Differentiator:** $v_{out}(t) = -RC \frac{dv_{in}(t)}{dt}$ (Replace $R_{in}$ with Cap).

### S&S - Z-Transform Basics
*   **Definition:** $X(z) = \sum x[n] z^{-n}$
*   **Unit Circle:** Maps to the $j\omega$ axis. Boundary of stability.
*   **Stability Rule:** All poles must satisfy $|z| < 1$ (Inside unit circle).
*   **Core Pair:** $a^n u[n] \iff \frac{z}{z-a}$, ROC: $|z| > |a|$.
*   **Delay Property:** $x[n-1] \iff z^{-1} X(z)$. ($z^{-1}$ is a unit delay in digital hardware).
