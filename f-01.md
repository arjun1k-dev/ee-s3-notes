## Day 1: The DC Foundation (ECA)

Most students treat KVL and KCL as "rules to memorize." In industry, they are the physical laws of the universe applied to lumped elements.

### 1.1 The Physics of KCL and KVL

**Kirchhoff's Current Law (KCL):** The algebraic sum of currents entering a node is zero.
*   **The Physics:** Law of Conservation of Charge. Charge cannot be created or destroyed. If 5 electrons flow into a node per second, exactly 5 electrons must flow out. A node is not a battery.
*   **The Math:** $\sum I_{in} = \sum I_{out}$

**Kirchhoff's Voltage Law (KVL):** The algebraic sum of voltages around a closed loop is zero.
*   **The Physics:** Law of Conservation of Energy. If you start at a point, walk around a loop, and return to the same point, your net change in potential energy must be zero.
*   **The Math:** $\sum V_{rises} = \sum V_{drops}$

### 1.2 The Passive Sign Convention (Crucial)
If you mess this up, every circuit you analyze will be wrong.

> **Engineer's Cheat Code: The "Tail-to-Head" Rule**
> 1. Assume a current direction ($I$) through a component.
> 2. Draw the voltage polarity across that component so that current enters the **positive (+)** terminal.
> 3. If you calculate $P = VI$ and the result is **positive**, the component is absorbing power (Passive). If **negative**, it is supplying power (Active, like a battery).

### 1.3 The "Delicacy" Problem: Node Voltage with Dependent Sources

This problem separates the engineers from the memorizers. We have a dependent source, which means the source value depends on a voltage or current *elsewhere* in the circuit. We cannot use simple series/parallel combinations. We must use the Node Voltage Method.

**The Circuit Setup (ASCII Art):**
We have a circuit with a ground node, a Node A, and a Node B.
*   A $10\text{V}$ independent DC source pushes current into Node A through a $2\Omega$ resistor.
*   Node A connects to Node B via a $4\Omega$ resistor.
*   Node B connects to ground via a $1\Omega$ resistor.
*   A Voltage-Controlled Voltage Source (VCVS) is connected between Node B and ground. Its value is $2 \times V_A$ (it forces a voltage of $2V_A$ into Node B, pushing current into the node).

```text
      +10V
       |
      [2Ω] R1
       |
       +--------+-------> Node A (Voltage = Va)
       |        |
      [4Ω] R2  (Dependent Source is elsewhere, just establishing nodes)
       |        |
       +--------+-------> Node B (Voltage = Vb)
       |
      [1Ω] R3
       |
      GND
```
*Wait, let's draw the actual circuit with the VCVS for maximum clarity.*

```text
        + 10V -
          |
         [2Ω] (R1)
          |
          o-------> Va
          |
         [4Ω] (R2)
          |
          o-------> Vb
          |        |
         [1Ω]      |   +------+
         (R3)      +---| VCVS |---+ (Source forces Vb = 2*Va)
          |            +------+   |
         GND                     GND
```
*Correction for mathematical simplicity, let's use a standard Node Analysis problem without the VCVS shorting things out, to show the methodology.*

**The Setup:**
Let's find the power dissipated by the $1\Omega$ resistor.

```text
  +12V -
   |
  [2Ω] R1
   |
   o----> Node A (Va)
   |    |
  [3Ω]  | [Dependent Current Source: 2*Vx pointing down]
  R2    |        |
   |    o        |
   o----> Node B | 
   |   (Vb)      |
  [1Ω]           |
  R3             |
   |             |
  GND-----------+
```
*Wait, text schematics for complex dependent sources are getting messy. Let's use a standard, elegant configuration.*

**The Elegant Setup:**
*   Node A is connected to a $12\text{V}$ source via a $2\Omega$ resistor.
*   Node A is connected to Node B via a $3\Omega$ resistor.
*   Node A is connected to ground via a Voltage Controlled Current Source (VCCS) pointing down. Its value is $2 \times V_x$.
*   Let $V_x$ be the voltage across the $3\Omega$ resistor, defined as $V_x = V_A - V_B$.
*   Node B is connected to ground via a $1\Omega$ resistor.

```text
         +12V -
           |
          [2Ω]
           |
           o-------------o (Node A)
           |             |
          [3Ω]           |  VCCS (points down to ground)
           |             |  Value = 2 * Vx
           o-------------o (Node B)
           |
          [1Ω]
           |
          GND
```

**The Grind:**
1.  **Identify Nodes:** Node A ($V_A$) and Node B ($V_B$). Ground is the reference.
2.  **Express $V_x$**: The VCCS depends on $V_x$, which is the voltage across the $3\Omega$ resistor.
    $V_x = V_A - V_B$
3.  **KCL at Node A:**
    Currents leaving Node A = 0.
    Through $2\Omega$ resistor: $(V_A - 12) / 2$
    Through $3\Omega$ resistor: $(V_A - V_B) / 3$
    Through VCCS (leaving): $2 \cdot V_x = 2(V_A - V_B)$

    Equation:
    $\frac{V_A - 12}{2} + \frac{V_A - V_B}{3} + 2(V_A - V_B) = 0$

4.  **KCL at Node B:**
    Currents leaving Node B = 0.
    Through $3\Omega$ resistor: $(V_B - V_A) / 3$
    Through $1\Omega$ resistor: $V_B / 1$

    Equation:
    $\frac{V_B - V_A}{3} + \frac{V_B}{1} = 0$

5.  **Solve the System:**
    Let's simplify Node B:
    $V_B - V_A + 3V_B = 0 \implies 4V_B = V_A \implies V_A = 4V_B$

    Now substitute $V_A$ into the Node A equation. Note that $V_A - V_B = 4V_B - V_B = 3V_B$.
    $\frac{4V_B - 12}{2} + \frac{3V_B}{3} + 2(3V_B) = 0$
    $2V_B - 6 + V_B + 6V_B = 0$
    $9V_B - 6 = 0 \implies 9V_B = 6 \implies V_B = \frac{6}{9} = \frac{2}{3} \text{ V}$

    If $V_B = 2/3 \text{ V}$, then $V_A = 4(2/3) = 8/3 \text{ V}$.

**The Joy of Understanding:**
We just solved a circuit with a dependent source without breaking a sweat. By systematically writing KCL at each node and expressing the dependent variable ($V_x$) in terms of node voltages, the complex circuit collapses into simple linear algebra.

### 1.4 Thevenin's Theorem: The Art of Collapse

Thevenin's theorem states: **Any linear circuit containing independent and dependent sources can be replaced by an equivalent circuit consisting of a single voltage source ($V_{th}$) in series with a single resistance ($R_{th}$).**

> **Physical Reality Callout:**
> Imagine a complex circuit board. You are asked to connect a new motor to two specific terminals on that board. You don't care about the thousands of transistors inside the board; you only care about "What voltage will it provide?" and "What is its internal resistance?". Thevenin says you can treat that entire complex board as a simple battery and a single resistor. It is the ultimate abstraction.

**The Rules of Thevenin:**
1.  **Find $V_{th}$:** Remove the load. Calculate the Open-Circuit Voltage across the terminals. $V_{th} = V_{oc}$.
2.  **Find $R_{th}$ (Method 1 - Independent Sources Only):** Turn off all independent sources (Voltage sources $\to$ Short Circuit, Current sources $\to$ Open Circuit). Calculate the equivalent resistance looking into the terminals.
3.  **Find $R_{th}$ (Method 2 - Dependent Sources Present):** If there are dependent sources, you CANNOT use Method 1. You must use the **Test Source Method**. Turn off all independent sources. Connect a $1\text{V}$ (or $1\text{A}$) test source at the terminals. Calculate the current it draws (or voltage it produces). $R_{th} = \frac{V_{test}}{I_{test}}$.

### 1.5 The "Delicacy" Problem: Thevenin with a Dependent Source

This is the classic exam question. It tests your KCL, your understanding of dependent sources, and your Thevenin mastery.

**The Setup:**
We want to find the Thevenin equivalent of the following network looking from terminals A and B.

```text
         + 10V -
           |
          [1Ω] R1
           |
           o-------------o (Terminal A)
           |             |
           |             |  Dependent Source: 
          [2Ω]           |  Current Controlled Voltage Source (CCVS)
           |             |  Value = 2 * Ix (Volts)
           |             |
           o-------------o (Terminal B)
           |
          GND
```
Where $I_x$ is the current flowing downwards through the $2\Omega$ resistor.

**Step 1: Find $V_{th}$ (Open-Circuit Voltage)**
Since terminals A and B are open, no current flows out of the terminal.
Let's do KCL at Node A.
*   The $10\text{V}$ source pushes current through $R_1$ to A.
*   Current flows from A through $R_2$ to ground. Let's call this $I_x$.
*   The CCVS is in parallel with the terminals. Since the terminals are open, the CCVS is essentially an open circuit for current flow *out* of A? No, wait. A voltage source (even dependent) allows current to flow *through* it. But if Terminal A and B are open, the only path for current is through the CCVS.

*Wait, let's draw it correctly. The CCVS is between A and B. The $R_1$ and $R_2$ are in series to ground.*

```text
      +10V -
        |
       [1Ω] R1
        |
        o-------> (Terminal A)
        |
       [2Ω] R2   (Current down through R2 is Ix)
        |
        o-------> (Terminal B) 
        |
       GND

Wait, where is the CCVS? Let's put the CCVS in series or parallel? 
A classic topology: The CCVS is between Terminal B and Ground.
```

Let's use the most standard university topology:
*   A $12\text{V}$ source in series with a $4\Omega$ resistor ($R_1$).
*   This connects to Node A.
*   A $2\Omega$ resistor ($R_2$) connects Node A to Node B.
*   A Dependent Voltage Source is connected between Node B and Ground. Its value is $3 V_x$, where $V_x$ is the voltage across $R_2$ (Node A to Node B).
*   We want the Thevenin equivalent at Node A and Ground.

```text
         +12V -
           |
          [4Ω] R1
           |
           o------------> (Node A / Terminal A)
           |
          [2Ω] R2   (Voltage across R2 is Vx)
           |
           o------------> (Node B)
           |              
           |   Dependent Voltage Source (3*Vx)
           |   + -
           |   |
          GND GND
```

**The Grind:**

**1. Find $V_{th}$ (Open Circuit Voltage at A to Ground)**
Since terminals A and Ground are open, no current flows out of Terminal A. Therefore, the current through $R_1$ must equal the current through $R_2$.
Let current $I$ flow clockwise.
$I = \frac{12 - V_A}{4} = \frac{V_A - V_B}{2}$

Also, the dependent voltage source fixes $V_B$:
$V_B = 3 V_x = 3 (V_A - V_B)$
Let's solve for $V_B$ first:
$V_B = 3V_A - 3V_B \implies 4V_B = 3V_A \implies V_B = 0.75 V_A$

Now substitute $V_B$ into the current equation:
$I = \frac{V_A - 0.75V_A}{2} = \frac{0.25V_A}{2} = 0.125 V_A$

Equate the two expressions for current:
$\frac{12 - V_A}{4} = 0.125 V_A$
Multiply both sides by 4:
$12 - V_A = 0.5 V_A$
$12 = 1.5 V_A$
$V_A = \frac{12}{1.5} = 8\text{V}$

So, the Thevenin Voltage $V_{th} = V_A = 8\text{V}$.

**2. Find $R_{th}$ (Test Source Method)**
Since there is a dependent source, we must use the test source method.
*   Turn off the $12\text{V}$ source (replace with short circuit to ground).
*   Connect a $1\text{A}$ current source from Ground to Terminal A (injecting $1\text{A}$ into Node A).

```text
   (Short to GND) 
     |
    [4Ω] R1
     |
     o------------> (Node A / Terminal A)  <--- 1A injected here
     |
    [2Ω] R2   (Vx)
     |
     o------------> (Node B)
     |              
     |   Dependent Voltage Source (3*Vx)
     |   + -
     |   |
    GND GND
```

We need to find $V_A$ when $1\text{A}$ is injected. Then $R_{th} = \frac{V_A}{1\text{A}} = V_A$.

KCL at Node A: Injected current = Current to GND via R1 + Current to Node B via R2.
$1 = \frac{V_A - 0}{4} + \frac{V_A - V_B}{2}$  (Since the 12V source is shorted, the top of R1 is 0V).

We still have the constraint $V_B = 0.75 V_A$.
Substitute $V_B$:
$1 = \frac{V_A}{4} + \frac{V_A - 0.75V_A}{2}$
$1 = 0.25 V_A + \frac{0.25V_A}{2}$
$1 = 0.25 V_A + 0.125 V_A$
$1 = 0.375 V_A$

$V_A = \frac{1}{0.375} = \frac{1000}{375} = \frac{8}{3} \approx 2.667\text{V}$

Since $R_{th} = \frac{V_A}{1\text{A}}$, then $R_{th} = \frac{8}{3}\Omega \approx 2.67\Omega$.

**The Joy of Understanding:**
The Thevenin equivalent is an $8\text{V}$ source in series with an $2.67\Omega$ resistor. If you connect *any* load to Terminal A and Ground, you can now use a simple voltage divider. You have collapsed a circuit with dependent sources and multiple nodes into a single battery and a single resistor. That is power.

## Day 1: Hour 3 - Digital Number Systems

Digital electronics is highly scoring because it is pure logic. But to understand logic gates, you must understand the number systems they operate on.

### 2.1 Why 2's Complement is a Mathematical Masterpiece

In hardware, addition is cheap. Subtraction is expensive (it requires borrowing logic). Engineers wanted to build circuits that could *only* add, but still subtract.

How do you subtract using addition? You add the negative number.
In binary, how do you represent a negative number?

**Attempt 1: Sign-Magnitude**
Use the leftmost bit (MSB) as a sign. 0 = positive, 1 = negative.
$+5 = 0101$
$-5 = 1101$
*Problem:* If you add $+5$ and $-5$:
$0101 + 1101 = 10010$. Drop the carry, you get $0010$ ($+2$). Wrong. The hardware has to do complex checks to handle this.

**Attempt 2: 1's Complement**
Invert all bits.
$+5 = 0101$
$-5 = 1010$
*Problem:* If you add $+5$ and $-5$:
$0101 + 1010 = 1111$ ($-0$). There are two zeros ($+0$ and $-0$), which wastes space and requires "end-around carry" logic. Still annoying.

**Attempt 3: 2's Complement (The Winner)**
Invert all bits, then add 1.
$+5 = 0101$
$-5 = 1011$ (Invert: 1010, Add 1: 1011)

Let's add $+5$ and $-5$:
$0101 + 1011 = 10000$. Drop the carry, you get $0000$ ($0$). Perfect!

> **Physical Reality Callout:**
> 2's complement is brilliant because of **modular arithmetic**. An 8-bit register can only hold values from $00000000$ to $11111111$ (0 to 255). It wraps around.
> If you want to subtract 5 from 0, you just count backwards by 5: $255, 254, 253, 252, 251$. $251$ in binary is $11111011$, which is exactly the 2's complement of 5. The ALU (Arithmetic Logic Unit) doesn't know it's subtracting; it just adds and lets the bits overflow. The math elegantly handles the negativity.

#### The "Delicacy" Problem: Range Overflow
You have an 8-bit signed system (using 2's complement). The range is $-128$ to $+127$.
What happens if you add $+100$ and $+50$?
$+100 = 01100100$
$+50  = 00110010$
Sum  $= 10010110$

The MSB is now 1. In 2's complement, this means the number is negative.
Invert: $01101001$. Add 1: $01101010 = 106$. So the sum is $-106$.

**The Joy of Understanding:**
The hardware didn't break; it just wrapped around (like an odometer rolling over). The CPU has an "Overflow Flag" that detects this. The math of 2's complement naturally signals when the physical register is too small to hold the truth. Understanding overflow is how you debug embedded systems when your variables act weirdly.

## Day 2: AC Fundamentals & Boolean Logic

### 3.1 Phasors: Freezing Time

In DC, voltage is a constant number ($12\text{V}$).
In AC, voltage is a function of time: $v(t) = V_m \cos(\omega t + \theta)$.
If you try to do KCL on $v_1(t) + v_2(t) = v_3(t)$ where these are sinusoids of the same frequency, you are doing trigonometry. It is hell.

**Euler's Identity to the Rescue:**
$v(t) = \text{Re} \{ V_m e^{j(\omega t + \theta)} \} = \text{Re} \{ (V_m e^{j\theta}) e^{j\omega t} \}$

Notice that $e^{j\omega t}$ is common to *every* signal in the circuit (because linear systems don't change frequencies). We can factor it out and ignore it.
We are left with: $\mathbf{V} = V_m e^{j\theta} = V_m \angle \theta$.

This is a **Phasor**. We have frozen time. We converted a differential equation into a simple algebraic equation in the complex plane.

### 3.2 Impedance: The AC Resistance

Since we froze time, derivatives become multiplication:
$\frac{d}{dt} \implies j\omega$

*   Resistor: $v = iR \implies \mathbf{V} = \mathbf{I}R \implies Z_R = R$
*   Inductor: $v = L\frac{di}{dt} \implies \mathbf{V} = L(j\omega)\mathbf{I} \implies Z_L = j\omega L$
*   Capacitor: $i = C\frac{dv}{dt} \implies \mathbf{I} = C(j\omega)\mathbf{V} \implies \mathbf{V} = \frac{1}{j\omega C}\mathbf{I} \implies Z_C = \frac{1}{j\omega C}$

> **Engineer's Cheat Code:**
> Remember that $\frac{1}{j} = -j$.
> So $Z_C = \frac{-j}{\omega C}$.
> Inductors have $+j$ (positive imaginary). Capacitors have $-j$ (negative imaginary). They are mathematical opposites. This is why they cancel each other out in resonance.

### 3.3 The "Delicacy" Problem: AC Mesh Analysis

Let's solve an AC circuit using phasors. This is pure algebra, but the numbers are complex.

**The Setup:**
A $10\angle 0^\circ \text{V}$ source is in series with a $4\Omega$ resistor. This connects to a parallel combination of an inductor ($j3\Omega$) and a capacitor ($-j6\Omega$). We want the current through the capacitor.

Let's convert to a single loop using impedance combination.
*   $Z_1 = 4\Omega$
*   $Z_2 = j3\Omega$
*   $Z_3 = -j6\Omega$

$Z_{parallel} = Z_2 || Z_3 = \frac{(j3)(-j6)}{j3 - j6} = \frac{-j^2 18}{-j3} = \frac{18}{-j3} = \frac{18}{-j3} \times \frac{j}{j} = \frac{j18}{3} = j6\Omega$.

> Wait, check the math: $j3 - j6 = -j3$. $-j \times -j = -1 \times j \times j = -1 \times -1 = 1$? No, $j \times -j = -j^2 = 1$.
> Wait, $(j3)(-j6) = -18j^2 = 18$.
> So $Z_{parallel} = \frac{18}{-j3} = \frac{j18}{3} = j6\Omega$.
> Wait, an inductor ($j3$) in parallel with a capacitor ($-j6$). The total is $j6$ (inductive). Because the capacitor's reactance is larger in magnitude, the net effect is inductive. This is physically correct.

Total impedance $Z_{total} = Z_1 + Z_{parallel} = 4 + j6 \Omega$.

Total current from source: $\mathbf{I}_{total} = \frac{\mathbf{V}}{Z_{total}} = \frac{10\angle 0^\circ}{4 + j6}$.
Convert denominator to polar: $4 + j6 = \sqrt{4^2+6^2} \angle \tan^{-1}(6/4) = \sqrt{52} \angle 56.3^\circ \approx 7.21 \angle 56.3^\circ$.

$\mathbf{I}_{total} = \frac{10\angle 0^\circ}{7.21\angle 56.3^\circ} = 1.38 \angle -56.3^\circ \text{A}$.

Now, use the Current Divider rule to find current through the capacitor ($Z_3 = -j6$):
$\mathbf{I}_C = \mathbf{I}_{total} \times \frac{Z_2}{Z_2 + Z_3} = 1.38\angle -56.3^\circ \times \frac{j3}{j3 - j6}$
$\mathbf{I}_C = 1.38\angle -56.3^\circ \times \frac{j3}{-j3} = 1.38\angle -56.3^\circ \times (-1)$

Multiplying by $-1$ is the same as adding $180^\circ$ to the phase.
$\mathbf{I}_C = 1.38 \angle (-56.3^\circ + 180^\circ) = 1.38 \angle 123.7^\circ \text{A}$.

**The Joy of Understanding:**
We just solved a circuit with differential equations (because of L and C) using simple complex arithmetic. The phase angle of $123.7^\circ$ tells us the capacitor current is actually leading the source voltage, which makes sense because it's a capacitor, but the overall circuit is lagging (inductive). The math perfectly encodes the physical phase shifts.

### 3.4 Boolean Algebra: The Algebra of Logic

George Boole invented a mathematical system where variables are only True (1) or False (0). Claude Shannon later realized this perfectly describes a switch (open/closed).

**The Axioms (The Basics):**
*   **AND (Multiplication):** $A \cdot B$. True only if both are true. (Series switches)
*   **OR (Addition):** $A + B$. True if either is true. (Parallel switches)
*   **NOT (Inversion):** $\bar{A}$. True if A is false.

**The Golden Rules (Memorize These):**
1.  $A + 0 = A$ (Anything OR'd with 0 is itself)
2.  $A + 1 = 1$ (Anything OR'd with 1 is 1)
3.  $A \cdot 0 = 0$ (Anything AND'd with 0 is 0)
4.  $A \cdot 1 = A$ (Anything AND'd with 1 is itself)
5.  $A + A = A$ (Idempotent Law)
6.  $A \cdot A = A$
7.  $A + \bar{A} = 1$ (Complement Law - A switch and its opposite in parallel is always closed)
8.  $A \cdot \bar{A} = 0$ (A switch and its opposite in series is always open)

---

### 3.5 De Morgan's Theorems: The Duality of Circuits

This is the most important theorem in digital logic.

1.  $\overline{A \cdot B} = \bar{A} + \bar{B}$
2.  $\overline{A + B} = \bar{A} \cdot \bar{B}$

> **Physical Reality Callout:**
> De Morgan's Theorems tell us that **NAND gates are universal**. You can build ANY logic circuit using only NAND gates.
> *   An AND gate is just a NAND gate followed by a NOT gate (which is a NAND gate with tied inputs).
> *   An OR gate is just a NAND gate with inverted inputs (which are NAND gates).
      > In industry, chips are manufactured with arrays of identical NAND gates because it's cheaper to print a million identical transistors than to print different types of gates. De Morgan's theorem is the mathematical justification for manufacturing efficiency.

#### The "Delicacy" Problem: Proving Equivalence

Prove that $(A + B) \cdot (\bar{A} + \bar{B}) = A \oplus B$ (XOR gate).

**The Grind:**
Let's expand the left side using the Distributive Law:
$LHS = A(\bar{A}) + A(\bar{B}) + B(\bar{A}) + B(\bar{B})$

Apply Complement Law ($A\bar{A} = 0$ and $B\bar{B} = 0$):
$LHS = 0 + A\bar{B} + \bar{A}B + 0$
$LHS = A\bar{B} + \bar{A}B$

**The Right Side (Definition of XOR):**
$A \oplus B$ is true when A is 1 and B is 0, OR when A is 0 and B is 1.
$RHS = A\bar{B} + \bar{A}B$

Therefore, $LHS = RHS$.

**The Joy of Understanding:**
We just proved that an XOR gate—which usually requires a specific complex circuit—can be built using standard AND and OR gates. This means if a factory only has AND and OR gates, they can still build an XOR gate. Boolean algebra allows us to translate logic between different physical implementations.

Add these to your master A4 sheet.

### ECA - DC
*   **Voltage Divider:** $V_{out} = V_{in} \frac{R_2}{R_1 + R_2}$
*   **Current Divider:** $I_1 = I_{total} \frac{R_2}{R_1 + R_2}$ (Note: The current through $R_1$ uses $R_2$ in the numerator).
*   **Thevenin ($V_{th}$):** Open circuit voltage.
*   **Thevenin ($R_{th}$):**
    *   Dep sources OFF $\implies$ equivalent resistance.
    *   Dep sources ON $\implies R_{th} = \frac{V_{test}}{I_{test}}$.

### ECA - AC
*   **Impedance:** $Z_R = R$, $Z_L = j\omega L$, $Z_C = \frac{1}{j\omega C} = \frac{-j}{\omega C}$
*   **RMS Value:** $V_{rms} = \frac{V_m}{\sqrt{2}}$
*   **Admittance ($Y$):** $Y = \frac{1}{Z} = G + jB$ (Conductance + Susceptance)

### Digital - Boolean
*   **De Morgan 1:** $\overline{AB} = \bar{A} + \bar{B}$
*   **De Morgan 2:** $\overline{A+B} = \bar{A} \cdot \bar{B}$
*   **Absorption 1:** $A + AB = A(1+B) = A$
*   **Absorption 2:** $A + \bar{A}B = (A + \bar{A})(A + B) = A + B$
